# Statements Section Writing

## Truth Boundary

For ethics, consent, registration, data sharing, code availability, funding, conflicts of interest,
author contributions, acknowledgments, and AI-use disclosure, write only from confirmed statement
facts. Unknown identifiers or approvals must remain unknown; do not infer them from study type or
journal expectation.

Contract phrase: write only from confirmed statement facts.

## Source Hierarchy

Use this order:

1. user-provided documents or explicit user facts,
2. protocol, registry record, ethics letter, data dictionary, repository record, funder record,
3. confirmed Paper Framework Statement Plan,
4. target journal required wording,
5. generic placeholder with a precise blocker.

## Statement Plan

Before drafting, build or update:

| Statement | Source fact | Required wording | Status | Blocker |
|---|---|---|---|---|

Status values should be `confirmed`, `not_applicable`, `missing`, or `needs_user_evidence`. Do not
use `confirmed` for a fact that is only likely.

## Wording Rules

- Ethics: include approving body and approval identifier only when known.
- Consent: distinguish written informed consent, waiver, opt-out, deidentified retrospective data,
  or not applicable.
- Registration: include registry name and identifier only when verified.
- Data availability: state availability, restrictions, repository, accession, or request route from
  confirmed facts.
- Code availability: distinguish code used for analysis from manuscript-generation scripts.
- Funding/conflicts: preserve exact funder, grant, and conflict facts.
- AI use: disclose only the confirmed use category and do not imply authorship or analysis work that
  did not occur.

## Blocker Text

Use precise markers when the statement cannot be completed:

```text
[AUTHOR: provide ethics committee name and approval number]
[AUTHOR: confirm whether informed consent was obtained, waived, or not applicable]
[AUTHOR: provide trial registration registry and identifier]
```

Do not replace these with generic fluent wording before the facts exist.
