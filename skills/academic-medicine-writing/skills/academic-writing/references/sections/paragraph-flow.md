# Medical Paragraph Flow

Use this guide with any section-specific guide. It is for local prose shape, not for inventing
evidence.

## One Paragraph, One Message

One paragraph, one message. Each paragraph must have one job: context, gap, objective, method
choice, result, interpretation, limitation, or implication. If a paragraph has two jobs, split it.
If two adjacent paragraphs have the same job, merge or cut.

## Paragraph Skeleton

Use this order unless a journal structure requires otherwise:

1. topic sentence: state the paragraph's function or claim,
2. support: give the source evidence, citation, method detail, or result needed for the claim,
3. boundary: add denominator, uncertainty, timeframe, missingness, bias, or scope when relevant,
4. transition: connect to the next section move.

The key sentence should not be buried in the middle. A reviewer should be able to skim only the
first sentence of each paragraph and recover the section logic.

## Medical Claim Flow

Keep the claim components stable:

```text
population / setting -> exposure, intervention, index test, or predictor -> comparator or reference
standard -> outcome -> timeframe -> analysis population -> estimate and uncertainty -> limitation
```

Use only the components relevant to the study design. If a component is absent from source material,
weaken the sentence or mark the gap.

## Salience Rules

- Spend words on claim-bearing evidence, not on generic disease burden or routine study details.
- Put exhaustive variable lists, checklist inventories, search strings, coding rules, and subgroup
  rosters in tables or supplements unless the body needs one example for interpretation.
- Every number in prose must support a sentence-level point. Do not transcribe tables into prose.
- Use precise nouns instead of bare "this", "these", or "it" when ambiguity can change medical
  meaning.

## Reverse Outline Check

After drafting, write a one-line reverse outline:

| Paragraph | Role | Main message | Evidence source | Medical boundary |
|---|---|---|---|---|

Revise any paragraph whose role, message, or evidence source is unclear.
