# Section Writing Guide Index

Use this index when drafting or revising named sections in a medical manuscript. The article-type
profile owns section order and word budget; these section guides own how each section should be
written.

## Routing

| Section task | Load |
|---|---|
| paragraph-level flow, compression, or local polish | `references/sections/paragraph-flow.md` |
| title, structured abstract, Key Points, or summary box | `references/sections/title-abstract.md` |
| Introduction | `references/sections/introduction.md` |
| literature positioning, background, related work, prior-work comparison | `references/sections/literature-positioning.md` |
| Methods | `references/sections/methods.md` |
| Results | `references/sections/results.md` |
| Discussion, limitations, conclusion | `references/sections/discussion.md` |
| appendix, supplement, eTables/eFigures, extended methods | `references/sections/appendix-supplement.md` |
| Ethics, consent, registration, data sharing, funding, conflicts, author contribution, AI use | `references/sections/statements.md` |

Always pair the section guide with the selected reporting checklist and article-type profile. For
complete drafts, also use the confirmed Paper Framework, display plan, statement plan, and target
journal standard card when available.

## Layer Boundaries

- `_shared/article-types/` decides the default section list, order, article-specific budget, and
  compression priority.
- `_shared/checklists/` decides reporting evidence requirements by study design.
- `_shared/journal-standards/` decides target journal limits and required submission elements.
- `references/sections/` decides section-level writing logic, paragraph flow, evidence-safe
  phrasing, literature-positioning moves, and appendix/supplement anchoring.

If the layers conflict, follow this precedence: current official target-journal instructions, target
journal standard card, selected article-type profile, selected reporting checklist, section guide,
generic fallback.

## Minimum Section Gate

Before writing any section, identify:

1. the section's job in the manuscript,
2. the confirmed source evidence that supports it,
3. the selected checklist items it must satisfy,
4. the population, setting, timeframe, outcome, analysis population, and uncertainty terms that must
   remain stable,
5. the statements, citations, tables, or figures required by the section.

If an input is missing and would change medical meaning, ask or mark a precise blocker. Do not turn
unknown clinical facts into polished prose.
