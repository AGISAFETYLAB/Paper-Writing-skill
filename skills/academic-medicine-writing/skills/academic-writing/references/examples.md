# Medicine Representative Structures

Use representative papers only for structure and reporting logic, not prose copying.

| Checklist | Representative structure cue |
|---|---|
| CONSORT | randomized design, randomization/blinding, participant flow, outcomes, harms |
| STROBE | exposure/outcome definitions, confounding, bias, missing data, sensitivity analysis |
| PRISMA | protocol, search strategy, screening flow, risk of bias, synthesis |
| STARD | index test, reference standard, participant flow, accuracy estimates |
| TRIPOD / TRIPOD-AI | intended use, data source, predictors, validation, calibration, clinical utility |
| CARE | patient information, timeline, diagnostic assessment, intervention, follow-up |
| GATHER | data source inventory, inclusion/exclusion, modeling assumptions, uncertainty |
| CHEERS | perspective, time horizon, costs, effects, model assumptions, sensitivity analysis |
