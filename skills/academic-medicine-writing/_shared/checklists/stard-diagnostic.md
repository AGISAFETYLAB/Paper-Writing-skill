# STARD Diagnostic Accuracy Card

Structured item slots: `items/stard-2015.json`.

Use for diagnostic tests, screening tools, imaging/lab tests, and diagnostic AI evaluations.

## Required Evidence Slots

- intended use, target population, setting, eligibility
- index test and reference standard definitions
- test order, blinding, threshold selection
- participant flow and excluded cases
- 2x2 counts or enough data to derive them
- sensitivity, specificity, PPV, NPV, AUC with confidence intervals
- indeterminate/missing results
- harms or downstream consequences if relevant

## Expected Display Items

Participant flow, 2x2 table, accuracy estimates table, ROC curve if threshold performance matters.

## Common Blockers

Do not generalize diagnostic performance beyond the tested population or reference standard.
