# STROBE Observational Study Card

Structured item slots: `items/strobe.json`.

Use for cohort, case-control, and cross-sectional studies.

## Required Evidence Slots

- study design, setting, dates, eligibility criteria
- participant selection and flow
- exposure, outcome, confounder, and effect modifier definitions
- data sources and measurement reliability
- bias and missing-data handling
- study size rationale when available
- statistical models, adjustment set, sensitivity analyses
- descriptive data and outcome events
- limitations, generalizability, ethics/data access

## Expected Display Items

Cohort flow, baseline/descriptive table, adjusted association table, sensitivity table.

## Common Blockers

Do not use causal language unless identification assumptions and bias controls support it.
