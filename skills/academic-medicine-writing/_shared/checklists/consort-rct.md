# CONSORT RCT Card

Structured item slots: `items/consort-2025.json`.

Use for randomized controlled trials of treatment, prevention, vaccine, behavioral, or care-delivery
interventions.

## Required Evidence Slots

- trial design, allocation ratio, setting, eligibility criteria
- intervention and comparator details
- primary and secondary outcomes with prespecified timepoints
- sample size rationale
- sequence generation, allocation concealment, blinding
- statistical analysis population and missing-data handling
- participant flow, recruitment, baseline data
- effect estimates with confidence intervals
- harms and adverse events
- trial registration, protocol availability, ethics/consent

## Expected Display Items

CONSORT flow diagram, baseline table, primary/secondary outcome table, harms table.

## Common Blockers

Do not write randomization, blinding, sample-size, registration, protocol, or harms details unless
the source material provides them.
