# Research Letter / Brief Report Profile

Use this profile for short original-research formats such as Research Letter, Brief Report, Brief
Communication, or other concise article types selected by the target journal.

## Profile Boundary

This profile owns section structure and proportional budget only. Apply
`_shared/article-types/profile-boundary.md` for hard-default and deviation rules.

## Length Budget Source

Take absolute length from the active `target_length_budget`. Short article types are usually hard
bounded by words, display items, and references; do not expand them into full IMRaD manuscripts.

## Checklist / Study-Type Pairing

Pair with the design-specific checklist in compressed form when required by the target journal and
source evidence.

## Priority Contract

- Primary core: Methods.
- Evidence core: Results.
- Compress first: background, extended discussion, secondary analyses, and literature review.
- Core floor: the reader must still see design, population/data, key variables/outcomes, analytic
  method, primary result, uncertainty, and one bounded interpretation.

## Section And Proportional-Budget Reference

| Order | Candidate section | Share of main text | Section role |
|---|---:|---:|---|
| Front | Title | outside main budget unless target counts it | State the specific research focus without a subtitle overload. |
| 1 | Introduction / Objective | 10-15% | One short paragraph giving the question and why it matters. |
| 2 | Methods | 28-36% | Compactly define design, population/data, variables/outcomes, and analysis. |
| 3 | Results | 34-42% | Report the primary finding with denominator, effect/estimate, uncertainty, and only essential secondary evidence. |
| 4 | Comment / Discussion | 14-20% | Interpret the finding, state a key limitation, and avoid broad claims. |
| Back | References / data sharing / supplement if allowed | outside main text unless target counts it | Keep references and supplement within the target article-type rules. |

## Flexible Adjustment Notes

- If the target forbids headings, keep this structure internally but write continuous prose.
- Use no more displays than the target allows. A single table or figure should carry the main result.
- If the evidence needs extensive methods or sensitivity analyses to be credible, use a full original
  investigation profile instead.
