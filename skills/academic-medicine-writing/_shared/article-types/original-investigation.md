# Original Investigation / Article Profile

Use this profile for complete original empirical medical, clinical, biomedical, public-health, or
translational studies when the target article type is Original Investigation, Article, Analysis, or a
broad original-research category.

## Profile Boundary

This profile owns section structure and proportional budget only. Apply
`_shared/article-types/profile-boundary.md` for hard-default and deviation rules.

## Length Budget Source

Take absolute length from the active `target_length_budget`. Use this profile to allocate the main
text after the target journal or fallback budget is resolved.

## Checklist / Study-Type Pairing

Pair with the selected study checklist: CONSORT, STROBE, STARD, TRIPOD, CHEERS, GATHER, or mixed
checklist as justified by the source evidence.

## Priority Contract

- Primary core: Methods.
- Evidence core: Results.
- Compress first: broad background, secondary discussion, secondary analyses, and nonessential
  implementation detail that can move to supplement.
- Core floor: Methods and Results must remain detailed enough for design, participants/data,
  variables/interventions/tests, statistical methods, denominators, and primary findings to be
  independently evaluated.

## Section And Proportional-Budget Reference

| Order | Candidate section | Share of main text | Section role |
|---|---:|---:|---|
| Front | Title / structured abstract / Key Points when required | outside main budget unless target counts it | State design, question, main finding, and cautious implication. |
| 1 | Introduction | 10-14% | Establish clinical or biomedical problem, evidence gap, and objective/hypothesis. |
| 2 | Methods | 28-36% | Define design, setting, participants/data, exposures/interventions/tests, outcomes, bias handling, sample size, missingness, and statistical analysis. |
| 3 | Results | 30-38% | Report cohort/sample flow, baseline characteristics, primary/secondary findings, uncertainty, sensitivity analyses, and applicable harms. |
| 4 | Discussion | 16-22% | Interpret findings against evidence, explain mechanisms or implications cautiously, state strengths/limitations, and avoid causal inflation. |
| 5 | Conclusions | 2-4% | Give the supported takeaway without new claims. |
| Back | Declarations / References / Checklist / Supplement | outside main text unless target counts it | Provide required statements, checklist, detailed tables, methods, or source data. |

## Flexible Adjustment Notes

- If the target is Nature Medicine Article/Analysis, adapt the section labels to the journal-required
  order while preserving the same Methods/Results evidentiary load.
- If the study is a trial, diagnostic accuracy study, prediction model, health economics evaluation,
  or global-health estimate, prefer the design-specific profile unless the target article type
  explicitly requires a broader original-investigation structure.
