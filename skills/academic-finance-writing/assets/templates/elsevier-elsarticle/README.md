# Elsevier elsarticle 模板说明（中文）

Original upstream README: [README](README)

这个目录存放 Elsevier `elsarticle` LaTeX 模板资产，供 `academic-finance-writing` 在需要 Elsevier-style 投稿或草稿 shell 时引用。

保留上游原始 `README`、模板文件、`.bst` 文件和文档，不直接改写第三方资产内容。当前中文页只是为了让仓库浏览时优先看到中文说明。

常用文件包括：

- `elsarticle-template-num.tex`
- `elsarticle-template-harv.tex`
- `elsarticle-template-num-names.tex`
- `elsarticle-num.bst`
- `elsarticle-harv.bst`
- `elsdoc.pdf`

真实投稿前仍应核对目标 Elsevier 期刊的最新 Guide for Authors。
