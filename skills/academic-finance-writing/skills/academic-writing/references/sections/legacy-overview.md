# Finance Section Guidance

Compatibility entrypoint for older workflows. For new work, load
`references/sections/index.md` first, then the section-specific guide named there.

Layer rule: paper-type profiles decide section order and budget priority; finance-domain adapters
decide terminology, variables, and reviewer risks; method-family guides decide identifying
assumptions and evidence standards; this section layer decides how each section should argue.

Required section guides:

- `references/sections/index.md`
- `references/sections/paragraph-flow.md`

- `references/sections/introduction.md`

Literature Review As Story, Results Presentation, Theory And Model Section.
