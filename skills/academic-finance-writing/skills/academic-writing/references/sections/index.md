# Finance Section Writing Guide Index

Use this index for section drafting, section revision, and full-manuscript drafting. It is a routing
file, not a substitute for the relevant section guide.

## Layer Boundaries

- `_shared/paper-types/`: section order, structural role, compression priority, and budget floor.
- `_shared/checks/finance-domains.md`: finance vocabulary, variables, institutional facts, literature
  neighborhoods, and reviewer risks.
- `_shared/checks/identification-strategies.md`: method family, identifying assumptions, diagnostics,
  and evidence standards.
- `references/writing-craft.md`: contribution, belief update, results narrative, and
  title-abstract-introduction gates.
- `references/prose-style.md`: sentence rhythm, anti-AI wording, triangular structure, and finance
  prose habits.
- `references/sections/`: section-level rhetorical movement, readiness checks, and paragraph
  construction.

Paper Framework says what each section must contain. Section-level rhetoric stays in
`references/sections/`.

## Writing Readiness

Before drafting a section, confirm:

- section role from the selected paper-type profile;
- main claim and evidence boundary from the confirmed Paper Framework;
- source display, estimate, model proposition, or dataset fact that supports the section;
- relevant finance-domain adapter and method-family guide;
- whether the section can make causal language, predictive language, descriptive language, or only
  scoped qualitative language.

If any item is missing, write a compact blocking note instead of inventing content.
