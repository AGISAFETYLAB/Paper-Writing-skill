# Finance Venue And Statement Planning

Confirm target-journal instructions before submission-ready wording.

Plan early:

- data availability and vendor/licence constraints
- code availability when reproducibility is claimed
- sample construction and variable definitions
- conflicts of interest, funding, and financial relationships
- supplementary robustness tables
- appendix for model details and proofs when needed

Do not invent data access, vendor permissions, code repositories, or robustness results.

## Common Venue Families

| Venue family | What to verify |
|---|---|
| Finance Top 3 journals | manuscript guidelines, title page, anonymity, Internet appendix, disclosure, data/code |
| Economics journals | replication/data policy, appendix/proofs, identification standard, disclosure |
| Elsevier finance journals | guide for authors, source files, possible `elsarticle` workflow, data availability |
| INFORMS / management journals | supplemental material, data/code/reproducibility, model details |
| Finance conferences | annual call, anonymous PDF, abstract, JEL codes, author-info separation |
