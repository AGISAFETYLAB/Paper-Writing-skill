# Journal of Financial and Quantitative Analysis Author Instructions

Official source: https://jfqa.org/submissions/

Source snapshot checked: 2026-06-22. Re-check before real submission.

## Applicability

Use when the target is Journal of Financial and Quantitative Analysis or a Cambridge-hosted finance
journal workflow.

## Page Window

```yaml
target_page_window:
  min_pages: 35
  max_pages: 50
  source_type: field_convention_fallback
  source_url: https://jfqa.org/submissions/
  count_scope: anonymous text-searchable manuscript PDF
```

Use the page-window pass/fail statuses defined in `_shared/submission/submission-standards.md`.
Record `below min_pages` and `above max_pages` as blocking page-window states before any completion claim.
Current JFQA instructions specify PDF, margins, font, abstract, and spacing, but
no local hard page cap is recorded here.

## Required Shape

- JFQA submission requirements plus Cambridge author/contributor policies
- author affiliations, authorship, AI-use, and contributor information handled under Cambridge/JFQA
  policy
- figures, tables, accessibility material, and supplemental files checked against Cambridge Core
  author instructions when relevant

## Template Role

No official JFQA LaTeX template is bundled. Use the target author instructions and a neutral
finance journal shell unless the user provides an official template.

## Blocking Checks

- Check both the JFQA submission page and Cambridge author-instruction pages before submission.
- Do not claim compliance from citation style alone.
- Do not invent author/contributor or AI-use disclosures.
